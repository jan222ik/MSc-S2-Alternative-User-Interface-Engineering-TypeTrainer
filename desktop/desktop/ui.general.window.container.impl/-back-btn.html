@Composable()
internal fun BackBtn(height: Dp, enableBackBtn: Boolean, onBackBtnAction: () -> Unit, backHoverBtnText: String, surface: Color)
Content copied to clipboard

The back button shows an icon based on the state of enableBackBtn. On Click the onBackButtonAction is invoked. Additionally, the button shows the backHoverBtnText on hover.

Parameters

height

defines the maximum height of the component

enableBackBtn

if true -> the button is enabled and the Arrow Icon is shown otherwise -> the Dashboard icon is shown

onBackBtnAction

Invoked if button was clicked

backHoverBtnText

Text shown when Arrow Icon is hovered

surface

Color of button surface
